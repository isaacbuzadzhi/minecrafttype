WALK CHARLOTTE - EASY GITHUB UPDATE

1. Unzip walk-charlotte-easy-upload.zip on your computer.
2. Open isaacbuzadzhi/minecrafttype on GitHub.
3. Choose Add file, then Upload files.
4. Upload site.zip and netlify.toml together.
5. Commit the changes.

You do not need to upload any folders.
You do not need to delete the old Minecraft files.
The new netlify.toml ignores the old game.zip, world.zip, and site files.
